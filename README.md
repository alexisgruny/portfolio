Mon portfolio personnel développé avec Next.js et TypeScript.

- **Next.js**
- **TypeScript**
- **Tailwind CSS**

## 📦 Installation
```bash
git clone https://github.com/alexisgruny/portfolio.git
cd portfolio
npm install
npm run dev
```
